**Address Book using AVL Tree**

An Address Book application implemented in **C++** using an **AVL Tree** data structure to ensure efficient and balanced search operations.

---

**Features**
- Insert contacts
- Search contacts by ID
- Display contacts sorted by ID
- Visualize AVL tree structure

---

**Technologies Used**
- C++
- Object-Oriented Programming (OOP)
- AVL Tree (Self-balancing Binary Search Tree)

---

**Project Structure**
- `src/address_book_application.cpp` — main application source code  
- `testcases/testcases.txt` — input file containing initial contacts

---

**Notes**
- Duplicate IDs are not allowed
- AVL rotations are applied automatically to keep the tree balanced
